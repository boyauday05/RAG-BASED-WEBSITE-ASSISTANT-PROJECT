# RAG-BASED-WEBSITE-ASSISTANT-PROJECT
RAG web assistant that answers user queries using website content. Built with LangChain, LLaMA/Groq LLMs for response generation, ChromaDB for vector storage, and Hugging Face sentence-transformers for embeddings. The application uses BeautifulSoup, Requests, and Unstructured for web data extraction, and Streamlit for an interactive user interface.
